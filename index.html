How it works

Four steps.
One complete system.

01 — PROJECT

Describe your project

Application type, industry, throughput target, budget, environment, and safety requirements. Takes under 2 minutes.

02 — SELECT

Choose your components

Toggle every component your system needs — robot, actuators, sensors, PLC, EOAT, safety systems, conveyors.

03 — CONFIGURE

Spec each component

Fill targeted fields per component — payload, stroke, I/O count, sensor type. Only relevant fields shown per selection.

04 — GENERATE

Get spec + BOM instantly

Full spec sheet plus a structured bill of materials with live Misumi, McMaster, Digikey, and Grainger links. Export to .csv.

What's generated

Everything in one output.

OUTPUT 01

Component recommendations

2–3 real, purchasable models per component — FANUC, KUKA, ABB, Keyence, Siemens, Festo, Pilz and more — matched to your exact specs with price ranges and fit rationale.

OUTPUT 02

System integration notes

How every component interfaces with the others — communication protocols, wiring dependencies, software requirements, and PLC compatibility.

OUTPUT 03

Total cost estimate

Line-item breakdown for every component plus integration, programming, and commissioning. Full system budget range you can put in a client proposal today.

OUTPUT 04

Bill of materials NEW

Structured BOM table with every recommended component, quantity, and unit cost. Export as .csv and share with procurement immediately.

OUTPUT 05

Distributor links NEW

Every BOM line item links directly to Misumi, McMaster, Digikey, and Grainger search results. One click from spec to shopping — no copy-pasting part names.

OUTPUT 06

RFQ draft

8–10 ready-to-send technical questions to include when you contact system integrators and vendors — written in the language they expect.
